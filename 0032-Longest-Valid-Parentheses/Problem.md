# 0032. Longest Valid Parentheses

## Description

Given a string containing just the characters `'('` and `')'`, return the length of the longest valid (well-formed) parentheses substring.

## Example

**Input:**
s = "(()"
<br>
**Output:**
<br>
2
<br>
**Explanation:**
<br>
The longest valid parentheses substring is "()".

## Constraints

- `0 <= s.length <= 3 * 10^4`
- `s[i]` is `'('`, or `')'`.
